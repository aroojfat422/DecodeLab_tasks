# DecodeBot - Python GUI Chatbot

DecodeBot is a simple desktop chatbot built using **Python** and **Tkinter**. It provides predefined responses to common user inputs through a clean and beginner-friendly graphical user interface (GUI).

This project is ideal for beginners who want to learn GUI development, event handling, and basic chatbot logic in Python.

- Interactive chatbot interface

# Project #2 – Basic Classification Model

## Project Overview

This project demonstrates the fundamentals of supervised machine learning by building a basic classification model using a small dataset. The objective is to train a model that can classify data into different categories and evaluate its performance on unseen data.

## Objectives

* Load and explore a dataset.
* Perform basic data preprocessing.
* Split the dataset into training and testing sets.
* Train a classification model using a supervised learning algorithm.
* Evaluate the model using accuracy and other performance metrics.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Google Colab

## Workflow

1. Import the required libraries.
2. Load and inspect the dataset.
3. Separate features and target labels.
4. Split the dataset into training and testing sets.
5. Train a Decision Tree Classifier.
6. Make predictions on the test data.
7. Evaluate the model using Accuracy Score and Classification Report.

## Learning Outcomes

* Understanding supervised learning.
* Working with real-world datasets.
* Training and testing a machine learning model.
* Evaluating classification performance.
* Building a complete machine learning workflow.

## Results

The model successfully classifies the dataset and demonstrates the complete machine learning pipeline, from data preparation to model evaluation.
